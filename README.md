# java-lotto

## 기능 목록
- 사용자 입력
    - [ ] 구매할 금액 입력
    - [ ] 수동으로 구매할 로또 수 입력
    - [ ] 수동으로 구매할 로또 번호 입력
    - [ ] 당첨번호 입력
    - [ ] 보너스 번호 입력
    
- [ ] 로또 자동 생성

- 결과
    - [ ] 두개의 로또 비교
    - [ ] n 개의 로또 비교
    - [ ] 수익 계산
    
## 예외
- 구매할 금액 입력시
    - [ ] 숫자가 아닌 문자 입력
    - [ ] 1000원 이하의 금액 입력
